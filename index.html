Secure File Storage




Cloud computing is gaining significant attention, however, security is the biggest hurdle in its wide acceptance, because a number of customers are sharing the same cloud. This project aims at designing a new security method by using a hybrid cryptography system, for data security in the cloud and to protect data from unauthorized access or hackers in cloud at the time of data transmission by encrypting the user data. Confidentiality and Authentication are the main focus of our project. This project will also include Secure Hash Algorithm for data integrity and will ensure Secure File Transfer of data over the internet.
